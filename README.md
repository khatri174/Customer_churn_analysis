# 📊 Customer Churn Analysis | Python & SQL

## 📖 Overview
Customer churn is a critical problem for businesses. This project analyzes customer behavior to identify patterns and key factors that lead to churn, helping businesses improve retention strategies.

---

## 🎯 Objectives
- Understand customer churn patterns  
- Perform data cleaning and preprocessing  
- Conduct Exploratory Data Analysis (EDA)  
- Calculate important business KPIs  
- Provide actionable business insights  

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy)  
- SQL (Data extraction & querying)  
- Matplotlib & Seaborn (Visualization)  
- Jupyter Notebook  

---

## 📂 Dataset Description
The dataset contains customer-related features:

- Churn (Target Variable)  
- AccountWeeks  
- ContractRenewal  
- DataPlan  
- DataUsage  
- CustServCalls  
- DayMins  
- DayCalls  
- MonthlyCharge  
- OverageFee  
- RoamMins  

---

## 🔍 Analysis Performed

### ✔ Data Cleaning
- Removed missing and inconsistent values  
- Converted data types  

### ✔ Exploratory Data Analysis
- Distribution of churn vs non-churn customers  
- Feature-wise impact analysis  

### ✔ KPI Analysis
- Churn Rate  
- Average Monthly Charge  
- Customer Service Call Trends  

### ✔ Correlation Analysis
- Identified relationships between variables  

---

## 📊 Key Insights
- Customers with high customer service calls are more likely to churn  
- Contract renewal significantly reduces churn  
- Higher monthly charges increase churn probability  
- Overage fees impact customer dissatisfaction  

---

## 📈 KPIs
- Overall Churn Rate  
- Average Revenue per Customer  
- Customer Retention Rate  
- Service Usage Patterns  

---

## 💡 Business Recommendations
- Improve customer support services  
- Offer incentives for contract renewal  
- Optimize pricing strategies  
- Identify high-risk customers early
